ParkMe
Find an available car park in real time

- HTML5 - CSS3 - JQUERY - JAVASCRIPT - GOOGLE MAPS API - GITHUB - ADOBE PHOTOSHOP - ADOBE ILLUSTRATOR -

ParkMe is a system designed to help users find car parks available in real-time, saving time by cutting out the time spent driving around busy streets to find a car park.

Using real-time data to guide users to an available car park, users can search for parks in any location, choose their preferred park and receiving directions with the choice of voice automated guidance.

New driver or experienced driver, passenger or co-pilot, ParkMe is designed to be used by everyone to facilitate the search for a car park in any location.
